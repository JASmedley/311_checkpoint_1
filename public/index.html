Checkpoint 1: CRUD App

Overview

This app manages users for a database. You can create, review, update, and delete users.

Routes

  • GET /users: Lists the entire array of users from the data/index.
  • GET /users/:id: Shows the user that matches the passed-in id.
  • POST /users: Creates a new user and adds it to the current array of users.
  • PUT /users/:id: Updates one user in the array based on the passed-in id.
  • DELETE /users/:id: Deletes one user in the array based on its id
