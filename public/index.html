Checkpoint 1: CRUD App

Overview

This app manages recipes for a website. You can create, review, update, and delete recipes.

Routes

  • GET /recipes: Lists the entire array of users from the data/index.
  • GET /recipes/:id: Shows the user that matches the passed-in id.
  • POST /recipes: Creates a new user and adds it to the current array of users.
  • PUT /recipes/:id: Updates one user in the array based on the passed-in id.
  • DELETE /recipes/:id: Deletes one user in the array based on its id
